High Noon

My first experience with any kind of game development. High Noon is a tabletop board game that pits four players in a survival scenario where they must collect resources and build shelter. The project was done as part of Shanghai American School's Innovation Institute project-based learning program, and is intended to analyze the tendencies of people to collaborate or compete with others when placed in situations where both outcomes have benefits.

This project was a group effort alongside classmates Luca Lee, Leonard Lee, and Tori Li. With respect to team responsibilities, I structured the core gameplay loop and came up with most of the game's mechanics.

As one can tell, the game and its content are "heavily inspired" by a certain movement shooter that came out during that time period (please don't sue any of us Blizzard we really liked Overwatch)

Slimland

What started as a personal project for working with Unity eventually became a small submission to my first game jam. Slimland is a platformer game where you play as a happy blue slime. This was my first solo game project, done with the intent of developing skills in all aspects of game development. As such, all assets in this project are self-made. A short version consisting of two levels was submitted to Devtober 2020, and I intend to continue working on it in my own time.

The version submitted for Devtober 2020 may be found here on my itch.io

Stop the Missiles

One of the assignments for my Game Development I course. "Stop the Missiles" is a shooter game that takes inspiration from Space Invaders and Missile Command, requiring players to survive an alien onslaught while defending their base.

Stop the Missiles was created in GameMaker Studio 2, the primary software used for the course.

Slimrun

Another assignment for my Game Development I course. Slimrun is a runner game that uses assets from Slimland to make a sort of "spin-off". The game involves dodging hazards and shooting enemies to make it to the end of the game.

Slimrun was created in GameMaker Studio 2. Slimrun can be downloaded and played here on my itch.io

Project Mindswipe

The big final project for my Game Development I course. Project Mindswipe is a stealth game which has players sequentially break into various locations to steal something, and then break out of those same locations after the item is taken. Project Mindswipe is my first experience with "proper" video game development, as we created, developed, and communicated the game to our professor and graders over a specified period of time, before concluding with a presentation.

This project was a group effort alongside Drake Miller, Lance Davis, Ram Venkatesh, Dana Ravago and Vinnie Colosimo, and was created in GameMaker Studio 2.With respect to team responsibilities, I designed each level's rooms and room mechanics, and wrote the random level generation algorithm to create levels with those rooms.
